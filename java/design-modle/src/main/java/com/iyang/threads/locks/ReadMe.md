####  知识点

- JMM : Java 内存模型, 用于屏蔽掉各种硬件和操作系统的内存差异，以实现让java程序在各种平台下都能达到一致的并发效果.  JMM 规范了Java虚拟机与计算机内存是如何协同工作的.
- volatile
- synchronized
- lock
- reentrantLock
- reentrantReadWriteLock
- AbstractQueuedSynchronizer
- CountDownLatch
- Semaphore
